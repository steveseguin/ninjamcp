# VDO.Ninja MCP

`@vdoninja/mcp` is a Model Context Protocol bridge built on VDO.Ninja WebRTC data channels.

It enables:

- private bot-to-bot messaging
- reliable P2P file transfer
- shared state sync for multi-agent workflows

## Quick Start

```bash
npm i @vdoninja/mcp @roamhq/wrtc
npx vdon-mcp-install
npx vdon-mcp-server
```

## Fun Agent Rooms

- Claude Code <-> Codex CLI helper loops
- OpenClaw <-> OpenClaw specialist bot collaboration
- Encrypted multi-agent brainstorming rooms
